Module Abonnement & Fidélité
Ce module permet la gestion complète des abonnements et du programme de fidélité de la plateforme ShowMatchGoOn. Côté abonnement, les utilisateurs peuvent souscrire à différentes offres (BASIC, PREMIUM) avec gestion des dates de début et de fin, tandis que les administrateurs disposent d'un tableau de bord complet pour créer, modifier et supprimer les offres disponibles. Côté fidélité, un système de points et de niveaux récompense l'engagement des utilisateurs sur la plateforme, avec une progression automatique entre les différents paliers. Le module est développé avec Angular 17 pour le frontend et Spring Boot avec MongoDB pour le backend, exposant des API REST sécurisées via JWT. Il intègre également une interface administrateur dédiée permettant le suivi en temps réel des abonnements actifs et des points de fidélité accumulés par les utilisateurs.
